# Commercial Usage

This project is free for **personal use**.

## Allowed

* Personal use by individuals
* Modifying and extending the software for personal use
* Internal evaluation by companies

## Not allowed without a commercial license

* Using the software to provide paid services (e.g. tax reporting for clients)
* Offering the software as part of a paid product or service
* Reselling or redistributing the software for commercial purposes
* Offering the software as a hosted or SaaS solution

## Commercial usage

If you are a business (e.g. accounting firm) and want to use this tool in production, you must obtain a commercial license.
